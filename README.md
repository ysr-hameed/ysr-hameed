### About Me

I'm a full-stack web developer who loves building modern, fast, and reliable web applications. I've worked with a variety of technologies — from backend APIs to frontend interfaces — and I enjoy every part of the process.

I don't stick to one stack. I use Node.js, React, Python, Laravel, and whatever fits best for the project. I also like integrating AI tools & APIs to make apps smarter and more efficient.

For me, clean code, usability, and performance always come first.
